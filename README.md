# Laboratorio DevOps – Kubernetes, Terraform & Ansible

Este repositorio contiene un laboratorio práctico de DevOps enfocado en:
- Infraestructura como código
- Configuración automatizada
- Kubernetes
- Observabilidad básica

## Componentes
- Proxmox VE
- Terraform
- Ansible
- Kubernetes
- Ingress NGINX
- HPA
- metrics-server

## Objetivo
Construir un entorno reproducible, documentado y realista,
orientado a prácticas DevOps.
